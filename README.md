# tf-ocp-infra

1. Clone this repo
2. Change variables file with your environment values. There is a sample variables file.
3. Sequence of commands
   
   ```bash
   
    # terraform init
    # terraform plan
    # terraform apply
   
   ```
   
 ## Helpful commands
 
 ```bash
 
  # terraform state list
  # terraform state rm <resource name>
 
 ```
